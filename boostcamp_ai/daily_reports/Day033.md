# Day 33. Object Detection, CNN Visualization

##### Object Detection?

- 이미지에 어떤 것들이 있는지 개체를 탐지하는 것
- 물체별 경계를 나타내는 Semantic Segmentation보다 한차원 더 복잡한 문제
- 물체가 (1) 어디에 있고 (2) 어떤 클래스에 해당하는 지 추론해야 하는 Task
